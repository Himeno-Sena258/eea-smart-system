# 工程教育专业认证智能服务系统

## 项目简介

工程教育专业认证智能服务系统是面向工程教育专业认证工作的综合服务平台。系统基于《工程教育认证办法》和《工程教育认证标准》，围绕专业人才培养方案、课程大纲、教学运行、课程考核、达成度评价、调查评价、持续改进和自评报告等核心业务，提供数据化、流程化、协同化的认证支撑能力。

系统主要服务于专业负责人、课程负责人、任课教师等角色，帮助高校专业建设团队对认证材料进行规范管理，对课程目标、毕业要求、培养目标之间的支撑关系进行量化分析，并辅助生成持续改进记录和自评报告内容。

## 核心功能

### 1. 人才培养方案管理

人才培养方案管理用于对专业培养方案进行结构化分解与版本维护，支持多个版本的培养方案管理，并可将同一培养方案应用到多个年级。

主要功能包括：

- 培养方案版本管理
- 培养目标管理
- 毕业要求管理
- 毕业要求指标点管理
- 课程体系设置
- 毕业要求与培养目标对应关系设置
- 课程支撑毕业要求关联矩阵设置
- 课程与毕业要求指标点关联关系设置

### 2. 课程管理

课程管理用于对课程大纲进行结构化管理，帮助课程负责人维护课程基本信息、课程目标、教学内容、考核方式和评价标准。

主要功能包括：

- 课程基本信息管理
- 课程资源管理
- 课程目标管理
- 课程目标对应毕业要求指标点管理
- 课程教学内容与课程目标对应关系管理
- 课程考核评价方式设置
- 课程考核评价标准设置

### 3. 教学运行管理

教学运行管理用于维护教学过程中的基础对象和教学任务分配关系。

主要功能包括：

- 学生信息管理
- 教师信息管理
- 课程教学班管理
- 课程与任课教师分配
- 教学任务管理

### 4. 课程考核评价

课程考核评价面向任课教师，按照课程负责人设置的考核评价方式和评价标准，对学生学习成果进行采集、评分和统计。

主要功能包括：

- 平时作业、考试试卷、课程报告等佐证材料采集
- 按课程考核方式录入学生成绩
- 按课程目标管理成绩数据
- 支持课程目标达成度计算的数据准备

### 5. 达成度评价

达成度评价用于根据评价模型自动计算课程目标达成度和毕业要求达成度，为认证评价和持续改进提供量化依据。

主要功能包括：

- 自定义达成度评价模型
- 课程目标达成度评价
- 毕业要求达成度评价
- 达成度结果统计与分析

### 6. 调查与评价

调查与评价通过问卷调查等方式，对培养目标、课程体系、教学环节、培养目标达成情况和毕业要求达成情况进行间接评价。

主要功能包括：

- 问卷设计与发布
- 问卷数据采集
- 培养目标间接评价
- 课程体系间接评价
- 教学环节间接评价
- 毕业要求达成间接评价

### 7. 持续改进

持续改进围绕课程目标达成情况和认证评价结果，对专业建设过程中的问题、分析、措施和改进记录进行闭环管理。

主要功能包括：

- 课程目标达成分析报告
- 培养目标持续改进记录
- 毕业要求持续改进记录
- 课程体系持续改进记录
- 课程大纲持续改进记录
- 教学环节持续改进记录

### 8. 自评报告

自评报告模块用于根据认证自评报告大纲进行任务拆分和协同编写，并支持部分内容根据系统数据自动生成。

主要功能包括：

- 自评报告大纲管理
- 报告任务分配
- 多人协同编写
- 数据自动填充
- 自评报告材料汇总

## 用户角色

- 专业负责人：负责人才培养方案、培养目标、毕业要求、课程体系和认证整体数据管理。
- 课程负责人：负责课程大纲、课程目标、课程考核方式和评价标准维护。
- 任课教师：负责课程考核评价、成绩录入、佐证材料上传和课程目标达成数据维护。
- 管理员：负责用户、角色、权限、基础数据和系统配置管理。

## 快速开始

### 安装前端依赖

```bash
npm run frontend:install
```

### 启动前端开发服务

```bash
npm run frontend:dev
```

默认访问地址：

```text
http://127.0.0.1:5173
```

### 构建前端

```bash
npm run frontend:build
```

### 启动后端服务

后端初始化为 Spring Boot 项目后，可在根目录执行：

```bash
npm run backend:dev
```

### 构建后端

```bash
npm run backend:build
```

### 根目录聚合命令

```bash
npm run dev
npm run build
npm run test
```

## 项目目标

本系统旨在提升工程教育专业认证工作的数字化管理水平，帮助专业建设团队实现认证数据统一管理、评价过程自动计算、改进记录持续跟踪、自评报告协同生成，从而提高认证准备效率和材料质量。
